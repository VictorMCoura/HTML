Exemplo 7 HTML - Elementos de Tópicos e Lista

Os elementos html que dividem o documento em Tópicos no html são representados pelas tags h e podem variar do 1 ao 5, sendo o menor o mais importante. Exemplo:

título - h1

título - h2

título - h3

título - h4

título - h5
título - h6

Elementos de lista - ul e ol

Os elementos de listas são aqueles que criam listas não ordenados - ul ou ordenados ol. Esses elementos são utilizados para listar itens no documentos html, e também são muito utilizados em menus.

Exemplo lista não ordenada ul:

  • Item 1
  • Item 2
    • Subitem 2.1
    • Subitem 2.2
    • Subitem 2.3
    • Subitem 2.4
      • Subsubitem 2.4.1
      • Subsubitem 2.4.2
      • Subsubitem 2.4.3
      • Subsubitem 2.4.4
  • Item 3
  • Item 4
  • Item 5

Exemplo de lista ordenada:

  1. Item 1
  2. Item 2
    • Subitem 2.1
    • Subitem 2.2
    • Subitem 2.3
    • Subitem 2.4
      • Subsubitem 2.4.1
      • Subsubitem 2.4.2
      • Subsubitem 2.4.3
      • Subsubitem 2.4.4
  3. Item 3
  4. Item 4
  5. Item 5
