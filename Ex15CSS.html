Flexgrid ou Grid Layout

O Flexbox e CSS Grid podem ser usados em conjunto. Você pode aplicar Flexbox a elementos dentro de um layout de grade CSS Grid, aproveitando o melhor dos dois mundos.

A escolha entre Flexbox e CSS Grid não se trata de um substituir o outro, mas sim de entender as necessidades específicas do seu layout e escolher a técnica que melhor atende a essas necessidades.
